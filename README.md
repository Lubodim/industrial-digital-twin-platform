# Industrial Digital Twin Platform

Industrial Digital Twin Platform is a local web-based system for managing a library of digital twins of industrial products.

The platform will provide:

- Digital twin library
- Product files and technical parameters
- Experiments linked to each digital twin
- User authentication and activity tracking
- Engineering and economic calculations
- External AI research agent
- Internal local AI engineer
- Local network deployment
- Protection of confidential industrial information

## Technology Stack

- Python
- Django
- PostgreSQL
- Docker
- Bootstrap
- Ollama
- External AI provider APIs
